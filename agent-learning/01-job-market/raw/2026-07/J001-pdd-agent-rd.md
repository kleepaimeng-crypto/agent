## 职责语义转述

- 设计任务规划、上下文、长短期记忆和工具调用等 Agent 核心模块，关注低延迟与可靠性。
- 建设 RAG 与交互环境，使用 Prompt 和工作流优化复杂业务中的效果、稳定性与可控性。
- 将 Agent/RAG 接入高并发分布式链路，使用缓存、消息队列和异步调度保障大流量稳定运行。
- 建立意图识别、输出过滤、权限和沙箱等安全机制，并观测成功率、延迟、Token 与业务指标。
- 验证 MCP、多智能体和长文本等新技术的业务价值。

## 要求语义转述

- 至少熟练掌握 Python、Java、Go 或 C++ 中一种语言，并具备数据结构、算法和工程实践基础。
- 理解 LLM、Agent、RAG 原理，做过 Prompt、Function Calling、MCP 或多智能体项目。
- 熟悉至少一种主流 LLM 应用框架；能说明完整项目从需求到落地的过程。
- 开源贡献、后训练经验、算法竞赛成绩及高并发后端经验属于加分项。

## 分类依据

应用层 Agent、RAG、后端、评估、安全和可观测性均为主职责，是方向 A 的高信号样本。
